## Adding New Tokens

VoidDex automatically supports any ERC-20 token that has liquidity on the integrated DEXes. If a token pair has a pool on any supported DEX, VoidDex can route swaps through it.

For tokens to be displayed in the UI token selector, they must be added to the curated token list. The list prioritizes:
- High liquidity tokens
- Tokens with verified contracts
- Popular DeFi and governance tokens
